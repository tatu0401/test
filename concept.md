# Концепт
## Жанр 
2D шутер
## Описание
Космическое пространство, главный герой - космический корабль, инопланетяне
## Механика
игрок может перемещаться по полю влево, вправо, вверх, вниз. Перемещение ограничено границами экрана
Может стрелять пулями для уничножения врагов, сталкиваться с объектами
## Сюжет
Главный герой - космический корабль, который встречает флотилию пришельцев, защищает Землю от вторжения
